# Snake Game

A classic Snake game built in Java using Swing.

## Features

- Classic Snake gameplay
- Golden apples that grant bonus points and extra length
- Custom snake color selection
- High score tracking
- Play Again button
- Start screen

## Controls

- ↑ Move Up
- ↓ Move Down
- ← Move Left
- → Move Right
